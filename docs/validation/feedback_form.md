# Pilot Feedback Form

Use after each debrief session (or end-of-day rollup).

## Quick ratings (1-5)
- Trust score ("I believe capture was complete/accurate") — **1-5**
- Cognitive load score ("This reduced my mental burden") — **1-5**

## Prompts
1. What part felt most helpful?
2. Where did the assistant miss context or tone?
3. Was follow-up questioning too shallow, too probing, or right-sized?
4. What should change before broader handout?

## Notes format
- Keep entries short and concrete.
- Tie each note to one specific moment in the flow when possible.
